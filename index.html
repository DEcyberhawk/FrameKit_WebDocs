1. Overview

FrameKit is a branding and sales platform that enables users to design Instagram storefronts, DM funnels, and cross-platform kits.

2. Departments & Access

  • IT Department
  • Marketing & Sales
  • Business Development
  • Switchboard
  • Accounts
  • HR
  • Legal
  • Branding & Design
  • Master Panel (Founder Access)

3. Key Innovations

  • Smart DM Pages
  • Auto Receipt Engine
  • Cross-Platform Sync (TikTok, Facebook, WhatsApp)
  • AI-Powered Brand Kit Generator
  • Social Market for Chats, Calls, File Sharing
  • Mini Shops with Affiliate Tools
  • Multilingual and Mobile Money Integration

4. Deployment & Architecture

Uses React (Vite) frontend, Node.js backend with Express, Render/Vercel hosting, modular API structure.

5. User Roles

  • Founder / Admin
  • Department Heads
  • Store Owners
  • Buyers / Users
  • Affiliates
  • Community Members

6. Support

PDF manuals, interactive web docs, and department-specific instructions available.
